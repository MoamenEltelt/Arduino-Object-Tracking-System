# Arduino Object Tracking System

## Description
This project is a smart object tracking system using Arduino, ultrasonic sensor, and servo motor.

## Components
- Arduino Uno
- HC-SR04 Ultrasonic Sensor
- Servo Motor
- Laser Module

## Features
- Detects objects using ultrasonic sensor
- Tracks object movement using servo motor
- Activates laser when object is detected
